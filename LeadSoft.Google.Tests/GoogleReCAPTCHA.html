Gerador de Token reCAPTCHA

Utilitário para gerar o token de resposta do Google reCAPTCHA v3 e usá-lo como valor do campo Response do DTOSiteVerifyRequest ao testar o adapter.

Insira a Site Key pública cadastrada no Google reCAPTCHA Admin Console. O token gerado é válido por cerca de 2 minutos.
Dica: Combine este token com a Secret Key do seu projeto para testar o método PostSiteVerifyAsync do adapter. A Secret Key nunca deve ser exposta no client-side.
